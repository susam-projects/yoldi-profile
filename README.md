## Description

A simple account viewer app.

## Installation

There's `.nvmrc` file with the required node version.

To install packages:

```shell
npm i
```

## Dev build

To start dev server:

```shell
npm run dev
```

The server will be accessible on:

```
http://localhost:3000
```

## Release build

To build the application:

```shell
npm run build
```
